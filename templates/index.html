Telegram Kurulum Adımları

  1. Telegram'da arama kısmına @BotFather yazın ve mesaj ekranını açıp /start tuşlayın.
  2. /newbot komutunu gönderin. Botunuza bir isim (örn. MidasSinyal) ve bot ile biten bir kullanıcı adı (örn. midas_sinyal_bot) verin.
  3. BotFather'ın size verdiği uzun HTTP API Token değerini yukarıdaki Bot Token API kutusuna yapıştırın.
  4. Ardından kendi sohbet ID'nizi öğrenmek için Telegram'da @userinfobot araması yapıp /start yazın. Size verdiği Id numarasını kopyalayıp yukarıdaki Chat / Kanal ID kutusuna yapıştırın.
  5. Telegram'da kendi oluşturduğunuz botun sohbet penceresini açıp /start yazarak botu aktif edin.
  6. Buradaki panelden Test Et butonuna basın. Telefondan bildirim geldiyse Kaydet'e basın.
